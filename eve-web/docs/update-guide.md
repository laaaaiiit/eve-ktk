# EVE v2 Update Guide

Для обновления используйте тот же единый скрипт, что и для установки:

```bash
curl -fsSL https://raw.githubusercontent.com/laaaaiiit/eve-ktk/main/install-one-line.sh | bash
```

Повторный запуск на уже установленной системе выполняет:
- обновление кода из Git;
- применение миграций БД;
- актуализацию runtime-директорий и сервисов.

Отдельный updater-скрипт больше не является основным путем обновления.
